쿠팡 물류센터 유사 시나리오 종합실습 데이터

이 데이터는 교육용으로 생성된 가상 데이터입니다. 실제 쿠팡 내부 데이터가 아닙니다.

파일 목록:
- coupang_orders_raw.csv: 원천 주문 로그. 결측, 중복, 이상치, 문자열 불일치 포함
- coupang_centers.csv: 물류센터 마스터
- coupang_daily_weather.csv: 일자·센터별 날씨 및 인력 데이터
- coupang_orders_YYYY-MM.csv: 월별 주문 CSV. 데이터 수집 및 concat 실습용

권장 실행 순서:
1. 쿠팡_물류센터_종합실습_데이터분석_과정.ipynb 실행
2. 10장~16장 순서대로 실습
3. outputs 폴더에 저장되는 정제 데이터와 KPI 결과 확인
